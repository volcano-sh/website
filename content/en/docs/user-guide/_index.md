# Volcano User Guide

Welcome to the Volcano User Guide. This comprehensive guide provides detailed instructions on how to configure and use various features of Volcano for Kubernetes workload management.

## Overview

The User Guide covers essential topics including:

- **Scheduler Configuration**: Learn how to configure the Volcano scheduler for different scenarios
- **Priority and Policies**: Configure priority classes and job policies
- **GPU Management**: Use GPU sharing, GPU number allocation, and Volcano vGPU features
- **Plugins**: Configure and use various Volcano plugins (MPI, PyTorch, Ray, SSH, SVC, and more)
- **Network Features**: Enable network topology-aware scheduling
- **Performance Tuning**: Optimize Volcano performance for your workload
- **Advanced Features**: DRA support, NUMA awareness, hypernode auto-discovery, and more

## Getting Started

Browse the topics in this section to learn how to configure and use specific Volcano features. Each guide provides step-by-step instructions with examples.
